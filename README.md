# qst-em-rs

## Introduction
This is a extension manager for the [qst](https://github.com/kumudiaorong/qst-grpc) written in rust. It is used to manage the extensions of the qst.
## Getting Started

### Get Release

No release yet.

### Build from source

#### Dependencies
First please install [rust](https://www.rust-lang.org/tools/install).

#### Get Source
```shell
git clone https://github.com/kumudiaorong/qst-em-rs.git
```

#### Build
```shell
cargo target build
```
